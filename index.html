ทำไมเราควรใช้ External CSS?

1: การเขียนแบบ External ช่วยให้โค้ดสะอาด

คำอธิบาย: การแยก CSS ออกจากไฟล์ HTML ทำให้ไฟล์ HTML มีหน้าที่จัดการแค่โครงสร้างและเนื้อหาเท่านั้น

เหตุผล: ลดความซับซ้อน และช่วยให้หาจุดที่ต้องแก้ไขได้เร็วขึ้น
2: สามารถใช้ไฟล์ CSS ร่วมกันได้หลายหน้า

คำอธิบาย: เขียนไฟล์ style.css เพียงไฟล์เดียว แล้วเชื่อมต่อกับหน้าเว็บกี่หน้าก็ได้

เหตุผล: Reusability เขียนครั้งเดียว ทุกหน้าเปลี่ยนตามทันที
3: ง่ายต่อการแก้ไขดีไซน์ในอนาคต

คำอธิบาย: เปลี่ยนสีธีมทั้งเว็บไซต์ได้เพียงแก้จุดเดียวในไฟล์ภายนอก

เหตุผล: Maintainability ปรับปรุงแบรนด์ได้ในเวลาไม่กี่วินาที
